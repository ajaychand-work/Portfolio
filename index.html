03 — projects

001 github ↗

Marketing Campaign Performance Analysis

Excel → MySQL → Python → Power BI

End-to-end analytics pipeline for an online grocery client. Processed 18 months of raw campaign data (26,000+ records across 3 Excel sheets) across 4 channels — Meta, Google, LinkedIn, and YouTube. Cleaned messy data in MySQL using REGEXP, STR_TO_DATE, and window functions (RANK, LAG, SUM OVER). Computed KPIs including ROI, ROAS, CTR, CVR, and CAC in Python using Pandas and Scikit-learn. Delivered an interactive Power BI dashboard for stakeholder reporting.

Key finding: YouTube was receiving significant spend at the lowest ROI across all channels. Recommended reallocation to Google — contributing to ~15% improvement in overall campaign ROI.
Python MySQL Power BI Pandas Scikit-learn EDA
